05 / Effects & Animations

Efectos

Gradient Text

Reimagined.

Clase: .text-gradient

CSS: background-clip: text con gradiente primary→accent

Glassmorphism

Contenido con blur de fondo

backdrop-filter: blur(16px) + borde rgba(255,255,255,0.06)

Glow Effect

Glow behind elements

box-shadow: 0 0 20px rgba(99,102,241,0.3) o blur absoluto

Scroll Reveal

Elementos aparecen al entrar en viewport

Clase: .reveal → .reveal.visible

Transition: 0.8s cubic-bezier(0.16, 1, 0.3, 1)

Transform: translateY(30px) → translateY(0)

Grain Overlay

SVG noise filter al 3% de opacidad. Agrega textura sutil al fondo.

Marquee

Scroll infinito horizontal para logos de partners

Clase: .marquee + .marquee-content

08 / Notas para Figma

Exportacion a Figma

Color Styles

  • Crear 6 color styles en Figma con los mismos nombres:
  • background, surface, primary, accent, text-main, text-muted
  • Agregar los gradientes como estilos separados

Text Styles

  • H1: Space Grotesk Bold 72px
  • H2: Space Grotesk Bold 48px
  • H3: Space Grotesk Semibold 20px
  • Body: Inter Regular 16px
  • Small: Inter Regular 14px
  • Caption: Inter Semibold 12px uppercase

Componentes

  • Crear como componentes de Figma:
  • Button/Primary, Button/Secondary, Button/Small
  • Card/Glass, Card/Glass-LG
  • Badge/Status, Badge/Tag
  • Icon/Container, Icon/Step
  • Avatar/Stack
  • Navbar/Default, Navbar/Scrolled

Estructura HTML

  • Cada seccion del HTML esta marcada con:
  • <!-- SECTION: Nombre -->
  • Secciones: Navbar, Hero, Marquee, Features, How It Works, Stats, Testimonials, CTA, Footer
